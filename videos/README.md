# TikTok Test Videos

Generated vertical MP4 creatives for organic TikTok testing.

- `rescue-rush-01-one-percent.mp4`: challenge hook, "Only 1% clear Level 2"
- `rescue-rush-02-wrong-tap.mp4`: failure hook, "One wrong tap ruins it"
- `rescue-rush-03-country-clash.mp4`: country-clash hook

Regenerate them with:

```bash
swift scripts/make_tiktok_videos.swift
```

The MP4 files are ignored by Git so the repository stays light.
