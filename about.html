Rescue Rush

Rescue Rush: Country Clash is a lightweight browser puzzle game where players clear stacked rescue icons, save pets, and compare country scores.

The game is built for quick mobile play. It runs in the browser, does not require an account, and stores local progress only on the player device.

Monetization

The current version uses mock rewarded ads for testing. If real ads are enabled later, rewarded ads will be optional and will only grant in-game help after the ad experience is completed.
