Local Debug Events

Events To Watch

start_level_2 means TikTok traffic reached the real challenge.

fail_level_2 means the game is hard enough to create ad demand.

ad_offer, ad_start, and ad_complete show rewarded ad demand.

challenge_again means replay intent.

Dashboard Setup

1. Create a free PostHog project.

2. Put the project key into posthogKey in index.html.

3. Put your TikTok Pixel ID into tiktokPixelId.

4. Create a PostHog funnel: page view → start_level_2 → fail_level_2 → watch_ad_click.
